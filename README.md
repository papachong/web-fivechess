# Web Fivechess (Vite + TS + Tailwind + Canvas)

## 开发
```bash
npm install
npm run dev
```

## 构建
```bash
npm run build
npm run preview
更多平台构建打包方法，详见：[BUILD_GUIDE.md](BUILD_GUIDE.md)
```

## 功能
- 15x15 棋盘，Canvas 渲染
- 点击落子，五子连珠即胜
- 支持排行榜、风格切换、体验友好，多平台支持

## 部署
- 构建产物在 `dist/`，可直接部署到 Vercel/Netlify/Cloudflare Pages 等静态托管。
