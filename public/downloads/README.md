## 📥 下载方式

### 🌐 Web 版本
无需下载！直接访问：
- **在线体验**：[https://fivechess.ruhooai.com/](https://fivechess.ruhooai.com/)
- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）

### 🖥️ 桌面应用

#### macOS（Intel & Apple Silicon）
- **文件**：`小miu仔五子棋-1.0.0-mac.dmg` 或 `.zip`
- **系统要求**：macOS 10.13+
- **安装步骤**：
  1. 下载 DMG 文件
  2. 双击打开
  3. 将应用拖拽到 Applications 文件夹
  4. 从 Launchpad 或 Applications 启动

#### Windows（x64）
- **文件**：`小miu仔五子棋 Setup 1.0.0.exe`
- **系统要求**：Windows 10+
- **安装步骤**：
  1. 下载 EXE 安装程序
  2. 双击运行安装向导
  3. 按提示完成安装
  4. 从开始菜单启动

### 📱 移动应用

#### iOS（iPhone & iPad）
- **来源**：Apple App Store
- **搜索**："小miu仔五子棋"
- **系统要求**：iOS 13+

#### Android
